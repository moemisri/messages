## Description

This is a basic Nest.js project aimed at practicing fundamental dependency injection (DI) and other core concepts. It provides a starting point for learning and experimenting with Nest.js framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
